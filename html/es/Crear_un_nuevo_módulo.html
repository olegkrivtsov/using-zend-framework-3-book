14. Crear un nuevo módulo

Cuando nuestra aplicación web crece en tamaño podemos encontrar beneficioso usar la característica de Zend Framework 3 llamada módulos. Las aplicaciones modulares consisten en piezas que tienen relativamente pocas dependencias una de otra. Esto permite instalar o remover módulos bajo demanda. En este capítulo aprenderemos lo que es un módulo de ZF3, cuando crear un nuevo módulo y como crearlo.
