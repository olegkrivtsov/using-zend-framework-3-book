8. Преобразование входных данных с помощью фильтров

В этой главе мы рассмотрим стандартные фильтры, которые можно использовать для веб-форм. Фильтр - это класс, который принимает входные данные, обрабатывает их и создает выходные данные. Кроме этого, мы покажем, как написать свой собственный фильтр.

В целом, фильтры можно использовать даже вне форм для обработки произвольных данных. Например, вы можете использовать фильтры в действии контроллера для преобразования данных, переданных в качестве переменных GET и/или POST, в определенный формат.

Zend\Filter Содержит различные классы фильтров.
Zend\InputFilter Реализует контейнер для фильтров/валидаторов.
