# Preface

"Using Zend Framework 3" est un livre gratuit et open-source sur la programmation
d'applications web avec Zend Framework 3.

## A propos de ce livre

Le livre "Using Zend Framework 3" est un livre open-source sur la création
de sites avec Zend Framework 3.
Ce livre utilise la licence [Creative Commons Attribution - Pas d’Utilisation Commerciale - Partage dans les Mêmes Conditions (CC BY-NC-SA 4.0)](https://creativecommons.org/licenses/by-nc-sa/4.0/deed.fr). 

ZF3 est un framework PHP moderne 
utilisé pour créer des sites web professionnels, évolutifs et sécurisés, faciles à tester et à maintenir. 
Le framework utilise les meilleures pratiques et modèles de conception inspirés de l'évolution 
du développement web. 
Cela inclut le modèle MVC (Model-Vue-Controller) permettant d'organiser et de standardiser une application.

Ce livre ne traite pas seulement du framework Zend, mais aussi des bibliothèques qui y sont étroitement liées.
Bien que Zend Framework 3 dispose d'un module dédié à l'accès aux bases de données, 
nous parleros d'une bibliothèque tierce appelée Doctrine ORM, 
Il s'agit d'une méthode standardisée orientée objet pour la gestion de base de données. 
Dans les exemples d'applications que nous allons aborder dans les chapitres de ce livre, 
nous utiliserons également la librairie CSS Bootstrap pour produire des pages Web stylisées et bien structurées.

## Public visé

Ce livre est destiné aux développeurs web impliqués dans le développement d'applications en PHP.
L'auteur s'efforce de commencer avec des choses simples qu'un débutant devrait
comprendre. Les choses compliquées arrivent à la fin dans les derniers chapitres. 
Ce qui en fait le premier livre sur Zend Framework accessible à un débutant.
Vous n'avez pas besoin d'être un gourou en design patterns pour comprendre le contenu du livre.

Pour lire et comprendre ce livre, vous devez avoir quelques bases de PHP.
Le mieux pour apprendre PHP est [son site officiel](http://php.net/) et sa [documentation](http://php.net/docs.php) en ligne
L'idéal serait que vous ayez des notions sur les requêtes HTTP, les variables GET et POST, 
les namespaces, les classes et les interfaces. 

Vu que PHP est étroitement lié à d'autres technologies web, il est également recommandé d'avoir les 
connaissances de base dans les domaines suivants:

 * HTML (Hyper Text Markup Language) -- utilisé pour créer des pages Web qui peuvent être affichées dans un navigateur Web.
 * CSS (Cascading Style Sheets) -- utilisé pour définir l'apparence d'une page Web, comme la taille de la police ou la couleur de fond.
 * JavaScript -- un langage de script utilisé pour rendre une page Web plus interactive.

Pour apprendre HTML, CSS and JavaScript, un bon point de départ sont les
[Tutoriaux W3Schools](http://www.w3schools.com/).

## Exemples et Code Source

Le contenu de ce livre est illustré avec des exemples de code. Chaque exemple est un site Web complet que vous pouvez installer et exécuter vous-même pour
voir Zend Framework 3 en action. Vous pouvez même utiliser les exemples comme base pour vos propres sites Web.

Tout le code source est stocké sur GitHub. Le code est disponible publiquement et vous pouvez télécharger entierement 
le dossier depuis [cette page](https://github.com/olegkrivtsov/using-zf3-book-samples).
Pour télécharger le dossier, cliquez sur le bouton *Clone or Download* qui se trouve sur la page 
(voir l'illustration ci-dessous).
