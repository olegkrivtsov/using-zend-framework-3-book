# Prefácio


"Usando o Zend Framework 3" é um livro gratuito e open-source sobre programação
Web com o Zend Framework 3.



## Sobre este Livro

O livro "Usando o Zend Framework 3" é um livro gratuito e de código aberto sobre a criação de
sites com o Zend Framework 3.
Este livro usa a licensa [Creative Commons Attribution-NonCommercial-ShareAlike](https://creativecommons.org/licenses/by-nc-sa/4.0/).

ZF3 é um framework de PHP moderno
destinado a construir sites profissionais, escaláveis ​​e seguros.
Esses sites são fáceis de testar e manter. O framework utiliza as melhores
práticas e padrões comuns de design inspirados na evolução do desenvolvimento web. 
Isso inclui o padrão * Model-View-Controller *, que permite que o código de um aplicativo seja organizado de maneira consistente e padrão.

Este livro não é apenas sobre o Zend Framework, mas também sobre o estreitamente relacionado
as suas bibliotecas. Embora o Zend Framework 3 tenha um componente dedicado para acessar o
banco de dados, neste livro usamos uma biblioteca chamada Doctrine ORM que é um 
Maneira padrão orientada a objeto para executar o gerenciamento de banco de dados. Nos exemplos de aplicativos,que criaremos nos capítulos deste
livro, O Framework CSS Twitter Bootstrap é usado para produzir páginas da Web bem desenhadas e organizadas.

## Público Alvo

Este livro destina-se a desenvolvedores da Web envolvidos no desenvolvimento de aplicativos Web em PHP.
O autor se esforça para começar com coisas simples que um iniciante deve
Compreender. Isto faz deste livro o primeiro livro sobre o Zend
Framework que é fácil de ler e entender para um novato.
Você não precisa ser um guru em design patterns para entender a maior parte do conteúdo.


Para ler e entender este livro, você precisa ter um conhecimento básico de PHP.
Um bom ponto para aprender PHP é [seu site oficial](http://php.net/)e o on-line
[documentação] (http://php.net/docs.php).
È recomendavel que você entenda HTTP Requests, GET e POST, variáveis, namespace, classes, e interfaces.

Como o PHP está intimamente relacionado a outras tecnologias Web, também é recomendável
Você tem alguma experiência básica no seguinte:

 * HTML (Hyper Text Markup Language) - usado para criar páginas da web que
   pode ser exibido em um navegador.
 * CSS (Cascading Style Sheets) - usado para definir a aparência
   de uma página, como o tamanho da fonte ou a cor do plano de fundo.
 * JavaScript - uma linguagem de script usada para fazer uma web
   página mais interativa.

Para aprender HTML, CSS e JavaScript, um bom ponto de partida é
[Tutoriais do W3Schools](http://www.w3schools.com/).
